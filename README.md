# note-taker

## Description

This app allows managers to type notes during their busy schedule and have them save to a website.  It is very easy to use and accepts any information typed into the text area.

## Installation

npm install express

## Resources

* Javascript
* Bootstrap
* Nodejs
* Expressjs
